Preprints in details

プレプリント

  1. Yoshito Ishiki, An interpolation of metrics and spaces of metrics, 2020, arXiv 2003.13277. ( link )
    筆者が距離関数のモジュライ空間の研究を始めた論文である．この論文では二倍性質や一様不連結性，また距離不等式に関してこれらを満たさないような距離関数がそのモジュライ空間の中でcomeagerであることを証明した． ついで，下の空間が第二可算で局所コンパクトハウスドルフの時に距離関数のモジュライ空間が完備距離付け可能であることを証明した．
  2. Yoshito Ishiki, Characterizations of Urysohn universal ultrametric spaces, arXiv:2303.17471, ( link )
    非アルキメデス的ウリゾーン普遍距離空間を，ある種の完全グラフ様の部分空間を含むことによって特徴付けた．この応用として，非アルキメデス的ウリゾーン普遍距離空間の有限個の直積や冪空間，対象積空間などが元の空間に等長であることを証明した．
  3. Yoshito Ishiki, A non-Archimedean Arens--Eells isometric embedding theorem on valued fields, arXiv:2309.06704 ( link )
    Arens--Eellsの埋め込み定理の非アルキメデス的類似と見做せる次の定理を証明した．任意の非アルキメデス的距離空間は，任意に付値体を与えるとその付値体の拡大体に等長に，閉集合として埋め込めてさらに元の付値体上で代数的独立であるようにできる．
  4. Yoshito Ishiki, Spaces of metrics are Baire, arXiv:2402.04565 ( link )
    距離化可能空間を与え，その距離化可能空間と同じ位相を誘導する距離関数全体の集合を考え，それに上限距離を導入することで位相空間にする． この空間は距離関数のモジュライ空間と考えることができ，最初に与えた距離化可能空間の上に展開可能な幾何学全体を反映していると考えられる． モジュライ空間の理論においては，モジュライのコンパクト性や完備性に注意が払われるが，この距離関数のモジュライは完備ではない． この論文では，そのようなモジュライ空間のコンパクト性，完備性の延長線として，距離関数のモジュライがベール空間であることを証明した．
  5. Yoshito Ishiki, Algebraic structures on non-Archimedean Urysohn universal metric spaces, in preparation.
    準備中
  6. Yoshito Ishiki, An isometric extensor of metrics, arXiv:2407.03030, ( link )
    距離化可能空間の閉部分集合を考える．ハウスドルフによって，この閉集合上の位相的距離は，空間全体に拡張できることが知られている．もちろん同じ位相を生成するという条件を保ったまま拡張ができるのである． この論文では，その拡張定理の発展として，同時拡張，つまり拡張を写像として取ることができ，さらに上限距離に関して等長であるようにもできることを証明した．
